Solución al problema del combo dinámico

Análisis del Problema

¿Por qué no funciona el combo dinámico?

  • Conflicto de funciones de guardado: Había dos funciones diferentes manejando el evento de guardar
  • Falta de sincronización: Los cambios en los radio buttons no activaban la actualización de categorías
  • ID incorrecto: El JavaScript buscaba un elemento con ID "categoria" cuando el ID real es "event-category"
  • Manejo incorrecto de eventos: No se estaban registrando correctamente los listeners para los cambios

Solución Implementada

  • Unificación del manejo de eventos: Una única función para guardar los eventos
  • Sincronización de categorías: Actualización inmediata al cambiar el tipo de evento
  • Corrección de IDs: Uso del ID correcto "event-category" para el combo box
  • Registro adecuado de eventos: Listeners correctamente configurados para los radio buttons
// Función corregida para actualizar categorías
function actualizarCategorias() {
  const tipoSeleccionado = document.querySelector('input[name="event-type"]:checked');
  if (!tipoSeleccionado) return;

  const tipo = tipoSeleccionado.value;
  const select = document.getElementById("event-category");

  select.innerHTML = '<option>Selecciona una categoría</option>';

  const categorias = tipo === "GASTO" ? categoriasGastos : categoriasCobros;
  categorias.forEach(cat => {
    const option = document.createElement("option");
    option.value = cat;
    option.textContent = cat;
    select.appendChild(option);
  });
}
